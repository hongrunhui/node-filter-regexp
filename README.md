## node-filter-regexp
#### 说明：筛选出框架源码里所有的的正则表达式，配合[https://regexr.com/](https://regexr.com/)，可以用于学习理解框架的正则。
* `regsResult`是筛选后的正则文件
* `regsSource`是本地想要被筛选的文件
* `npm instal`、`node index.js`输入源码cdn即可
